# rjd3production: Prepare for Production of Seasonal Adjustment with 'JDemetra+'

A comprehensive tool for setting up CVS-CJO data pipelines using
'JDemetra+' (version 3) and 'rjdverse'. This includes setting up a new
working environment, creating and selecting calendar regressors,
managing specifications (CJO and outliers) at the workspace level,
making a workspace crunchable, removing insignificant outliers, and
comparing workspaces.
